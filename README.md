# Real Estate Price Predictor

A machine learning project that predicts residential property prices based on key housing attributes such as area, bedrooms, bathrooms, furnishing status, and location. The project demonstrates an end-to-end data science workflow including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment.

## Key Highlights

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Multiple Regression Models
* Model Performance Comparison
* Feature Importance Analysis
* House Price Prediction Interface
* Model Deployment with Streamlit

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Joblib
* Streamlit

## Objective

The objective of this project is to develop a predictive model capable of estimating house prices accurately based on historical housing data and property characteristics. The project explores multiple machine learning algorithms and selects the best-performing model using regression evaluation metrics.

## Results

The final model achieves reliable price predictions and provides insights into the factors that most significantly influence property values.
